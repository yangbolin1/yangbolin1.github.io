# Camera Learning Lab

记录 Android Camera、CameraX、Camera2、图像处理、预览渲染和拍照流程的系统学习过程。

## 学习方向

### Camera 基础

* 相机预览流程
* 拍照流程
* 自动对焦、曝光、白平衡
* 图片旋转、裁剪、压缩

### Android CameraX

* Preview 预览
* ImageCapture 拍照
* ImageAnalysis 图像分析
* Lifecycle 绑定
* CameraX 实战问题记录

### Android Camera2

* CameraManager
* CameraDevice
* CameraCaptureSession
* CaptureRequest
* Surface / TextureView / ImageReader
* Camera2 打开相机完整流程

### 图像格式

* YUV
* NV21
* JPEG
* Bitmap
* ImageReader 数据解析

### Bug 排查

* 预览黑屏
* 预览变形
* 图片旋转错误
* 拍照崩溃
* 不同手机兼容性问题
